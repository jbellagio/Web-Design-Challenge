Summary: Latitude vs X

The purpose of this project was to design a series of web pages using html. In order to do so effectively, I was required to pull elements of this from bootstrap. I also needed to create a style sheet using css.


A crucial step is linking them together through a navbar, so you have access to all the pages, all of the time. These web pages currently demonstrate data that analyzes how a city's latitude influences aspects of its weather. These aspects include maxmimum temperature, cloudiness, humidity, and wind speed. Feel free to explore.
